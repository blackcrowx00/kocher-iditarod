🤖 How Live Data Works A GitHub Actions workflow (scrape-standings.yml) runs every 3 hours. It launches a headless browser, logs into iditarod.com with your Insider account credentials, scrapes the live standings, and saves them to standings.json in your repo. This page fetches that file automatically.

One-time setup: Go to your GitHub repo → Settings → Secrets and variables → Actions → click New repository secret and add:
  • IDITAROD_USERNAME — your iditarod.com email/username
  • IDITAROD_PASSWORD — your iditarod.com password

Then go to the Actions tab → click Scrape Iditarod Standings → Run workflow to test it immediately.
